My Projects

Netflix Clone

netflix-clone-rugg.webapp.com

Built a fully functioning clone of the popular Video Streaming platform Netflix. It was implemented using React.js, along with server and client-side rendering using a Redux template. The movies are fetched using the tmdb API. It also uses Firebase and Firestore for Authentication and Data Storage respectively along with a fully functioning payment process using the Stripe API.

AirBnb Clone

airbnb-clone-ka28cev75-rugg07.vercel.app

Built a fully functioning clone of the world-famous Website Airbnb. The website was built on Next.js that helps in server-side rendering. Details of the posted homes were done using a pre-built JSON template and the integration of the website is done using Tailwind CSS, along with Mapbox API to show the locations of the properties in real-time.

Blood Donation Website

blood-donation.com

With the ongoing Covid-19 I thought of making a static website to store the data of people who had donated blood to the latest date.

It was implemented using PHP & HTML, the database was made using MySQL and an Apache Server to host it locally. I then deployed it on AWS - EC2 instances along with it's storage in S3. The User details section was made using an existing data model from Kaagle.
